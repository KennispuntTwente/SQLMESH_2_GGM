# ggm

In deze map vindt je door Delft gegenereerde SQL-code ('ddl/')
welke gebruikt kan worden om de tabellen van het GGM aan te maken.

Deze SQL-code valt onder de licentie zoals beschreven in 'ddl/LICENSE'. 

In dit project maken wij een selectie uit de GGM DDL voor onze use-case,
specifiek voor tabellen die betrekking hebben op het sociale domein.
Deze selectie staat in de map 'selectie/'.
